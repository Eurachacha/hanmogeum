## 화면 구성

- 메인페이지
- 로그인/회원가입 페이지
- 상품 목록/상품 상세 페이지
- 장바구니 페이지
- 상품 구매/구매 확인/주문완료 페이지
- 마이페이지 > 주문내역 / 주문상세 페이지
- 마이페이지 > 사용자 정보 수정 페이지
- 관리자 페이지 > 상품 관리페이지
- 관리자 페이지 > 주문 관리페이지
- 나만의 차 찾기 페이지
